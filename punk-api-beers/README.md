# punk-api-beers

A small VUE app to learn about different beers. To search for a beer, use the search bar at the top.

## Project Setup
From github:

clone the project into a local repository with:
```sh
git clone {github link}
```

```sh
npm install
```

### Run the app

```sh
npm run dev
```

### View the UI

visit localhost{PORT} in the browser
